## 关于这个站点

此站点基于 _facebook_ 的 [Docusaurus](https://docusaurus.io) 搭建

站点支持 mdx 文件编写交互式的 react 组件，渲染在页面中

交互示例使用 [react-live](https://github.com/FormidableLabs/react-live) 实现，通过 _webpack_ 替换了底层编译逻辑

## 如何运行

```shell
cd apps/lesson-4-hooks

rushx start
```
