# 使用说明

## Step 1 安装 rush 工具

```shell
npm i @microsoft/rush -g --registry=http://registry.npm.taobao.org
```

## Step 2 安装依赖

```shell
rush install --to @sjtu-fe/toutiao-lite-web
```

## Step 3 运行 demo

```shell
cd apps/toutiao-lite-web && rushx dev
```

## Step 4 安装 vscode 插件

打开 vscode 根据右下角提示安装插件
